# Motorola-Sales-Dashboard
1. Mobile Sales Data Dashboard — Power BI

2. Project Title
Mobile Sales Performance Dashboard using Microsoft Power BI

3. Short Description
An interactive and dynamic Mobile Sales Data Dashboard built in Microsoft Power BI, designed to analyze and visualize smartphone sales performance across major Indian cities. The dashboard provides actionable business insights including revenue trends, brand comparisons, payment behavior, and customer satisfaction — all in a single-page view.

4. ech Stack
ToolPurposeMicrosoft Power BIDashboard creation & data visualizationMicrosoft Bing MapsCity-wise geographic sales mappingDAX (Data Analysis Expressions)KPI calculations & custom measuresPower Query (M Language)Data cleaning & transformationExcel / CSVSource data format

5. Features

📊 KPI Cards — Total Sales (₹769M), Total Quantity (19K), Transactions (4K), Average Order Value (₹40.11K)
🗺️ Interactive Map — City-wise sales distribution across India
📈 Monthly Trend Line — Total quantity sold across all 12 months
📱 Top Mobile Models — Bar chart ranking best-selling phones (iPhone SE, OnePlus Nord, Galaxy Note 20, etc.)
🏷️ Brand Performance Table — Apple, Samsung, OnePlus, Vivo & Xiaomi with sales & quantity
💳 Payment Method Donut Chart — UPI, Cash, Debit Card & Credit Card breakdown
⭐ Customer Ratings Chart — Distribution of 1–5 star ratings
📅 Day-wise Sales Trend — Identifies peak & low sales days (Saturday highest, Wednesday lowest)
🔽 Slicers/Filters — Filter by Month, Payment Method & Brand for deep-dive analysis
